# BGFree v2.0 产品需求文档

> 版本: v2.1  
> 日期: 2026-03-22  
> 状态: 已确认

---

## 一、项目背景

BGFree 是一个在线图片去背景工具，当前已完成 MVP 版本（v1.0），具备本地 AI 和云端 remove.bg 双引擎去背景能力。

v2.0 的目标是：**接入用户体系和支付系统，实现商业化运营。**

---

## 二、用户角色

| 角色 | 描述 |
|------|------|
| 未登录用户 | 未注册，只能使用本地去背景，云端不可用 |
| 免费用户 | Google 登录后，本地无限 + 云端 3 次（终身，试用期） |
| 付费用户 | 购买了积分包或订阅了月套餐，本地无限 + 云端按额度 |

---

## 三、定价方案

### 3.1 双模式定价

采用 **积分包（一次性购买）+ 月订阅（自动续费）** 双模式，覆盖不同消费习惯。

| 类型 | 套餐 | 价格 | 云端次数 | 单次成本 | 毛利率 |
|------|------|------|---------|---------|--------|
| 试用 | 注册免费 | $0 | 3 次（终身） | - | - |
| **积分包** | 小包 | $4.99 | 10 次 | $0.50 | 60% |
| **积分包** | 中包 | $12.99 | 30 次 | $0.43 | 54% |
| **积分包** | 大包 | $29.90 | 80 次 | $0.37 | 46% |
| **月订阅** | 基础 | $9.99/月 | 25 次/月 | $0.40 | 50% |
| **月订阅** | 进阶 | $19.99/月 | 60 次/月 | $0.33 | 40% |

> **API 成本**：remove.bg $0.20/次（3 次免费试用走免费 Key，不产生成本）

### 3.2 套餐对比

| 功能 | 未登录 | 免费(3次) | 积分包 | 月订阅 |
|------|--------|----------|--------|--------|
| 本地去背景 | ✅ 无限 | ✅ 无限 | ✅ 无限 | ✅ 无限 |
| 云端去背景 | ❌ | 3 次（终身） | 按购买额度 | 按月额度 |
| 背景色替换 | ✅ | ✅ | ✅ | ✅ |
| 下载 PNG | ✅ | ✅ | ✅ | ✅ |

### 3.3 盈亏分析

**免费试用成本**：3 次 × 注册用户数。3 个免费 Key 合计 150 次/月，超出部分走付费 API。

| 规模 | 试用API消耗 | 付费API成本 | 盈亏平衡转换率 | 10%转换率月利润 |
|------|-----------|-----------|-------------|--------------|
| 10 人 | 30次（免费Key覆盖） | $0 | **0%** | $13 |
| 100 人 | 300次（需付费Key） | $30+ | **~5%** | $51 |
| 1000 人 | 3000次 | $570+ | **~8%** | $320 |

> 工具类产品 10%+ 转换率为行业合理水平，商业模型成立。

## 五、功能需求

### 5.1 Google OAuth 登录

**已有，需调整：**

- 登录后首页显示用户头像和用户名
- 登录成功后跳回首页
- 未登录时点击云端处理，弹窗提示登录（而非直接跳 Google）
- 退出登录清除本地 session

### 5.2 用量计数

**云端处理时扣减逻辑：**

1. 未登录用户：云端不可用（弹窗提示注册获得 3 次免费试用）
2. 已登录免费用户：终身 3 次，扣完弹窗引导购买（D1 数据库记录）
3. 积分包用户：扣减积分余额，用完弹窗引导续购
4. 月订阅用户：按月重置额度，用完弹窗引导升级

**计数时机：** 云端处理请求发出时扣减（非处理成功后），防止超用。

### 5.3 个人中心

**入口：** Header 头像点击 → 下拉菜单 → 个人中心

**页面内容：**

| 模块 | 说明 |
|------|------|
| 用户信息 | 头像、用户名、Google 邮箱、注册时间 |
| 当前套餐 | 免费用户 / 积分包（剩余 X 次）/ 月订阅（X 次/月，到期时间） |
| 用量统计 | 已用 / 总额度，进度条展示 |
| 积分余额 | 积分包用户显示剩余次数 |
| 升级/续购 | 跳转定价页 |
| 付费记录 | 购买记录列表（时间、套餐、金额、状态） |
| 退出登录 | 清除 session |

### 5.4 定价页面

**入口：** Header "升级" 按钮 + 个人中心升级链接 + 用量不足弹窗

**页面结构：**

```

  标题: 选择适合你的方案
  副标题: 本地去背景永久免费，云端高质量处理按需购买

  ──── 一次性积分包 ────
  [小包 $4.99]    [中包 $12.99]    [大包 $29.90]
  10 次           30 次            80 次
  $0.50/次        $0.43/次         $0.37/次
  [购买]          [购买]           [购买] ★推荐

  ──── 月订阅（自动续费）────
  [基础 $9.99]    [进阶 $19.99]
  25 次/月        60 次/月
  [订阅]          [订阅]

```

**定价卡片交互：**
- 未登录 → 点击购买/订阅 → 先 Google 登录 → 再跳转支付
- 已登录免费 → 点击购买/订阅 → PayPal 支付
- 已有积分/订阅 → 显示当前状态，引导续购/升级

### 5.5 PayPal 支付

**支付流程：**

```
用户点击购买/订阅 → 选择套餐 → PayPal 支付 → 支付成功回调 → 更新 D1 用户额度 → 跳转成功页
                                                            ↓ 失败
                                                        跳转失败页（重试按钮）
```

**技术方案：**

- PayPal Checkout（网页端标准支付）
- API Route `/api/payments/create-order` — 创建订单
- API Route `/api/payments/confirm` — 支付成功回调
- API Route `/api/payments/webhook` — PayPal webhook 通知
- 积分包：一次性支付，到账即充入积分
- 月订阅：月付自动续费，用户可随时取消（当期有效，下月取消）

### 5.6 FAQ

**定价页底部展示，折叠/展开：**

| 问题 | 回答 |
|------|------|
| 本地去背景真的免费吗？ | 是的，永久免费，不限次数，无需注册 |
| 云端去背景和本地有什么区别？ | 本地在浏览器处理，速度快但质量一般；云端用专业 AI 处理，边缘更精细（头发丝、透明物体等） |
| 免费额度用完了怎么办？ | 注册后可获得 3 次云端免费体验，用完后可选择积分包或月订阅 |
| 积分包和月订阅怎么选？ | 偶尔用选积分包（用完为止），经常用选月订阅（更划算） |
| 积分会过期吗？ | 不会，积分永久有效 |
| 可以取消月订阅吗？ | 可以，个人中心随时取消，取消后当期仍可使用，下月降为免费 |
| 支持哪些支付方式？ | PayPal（支持信用卡、借记卡、PayPal 余额） |
| 我的图片安全吗？ | 本地模式图片不会上传；云端模式通过加密传输，处理完即删除 |

### 5.7 未登录转化引导

**场景：** 未登录用户选择云端模式时

**交互：**
1. 弹窗提示："注册后可获得 3 次云端免费体验"
2. 按钮：[Google 登录] [继续用本地模式]
3. 点 Google 登录 → 授权 → 跳回 → 自动开始云端处理

**场景：** 免费用户 3 次用完后

**交互：**
1. 弹窗提示："免费试用已用完，购买积分包或订阅月套餐继续使用"
2. 按钮：[查看套餐] [继续用本地模式]
3. 点查看套餐 → 跳转定价页

## 七、remove.bg API Key 管理

**轮换策略：**

```
云端请求 → 检查所有 API Key 本月用量 → 选剩余最多的 Key → 转发请求
                                          ↓ 全部用完
                                    走付费订阅 Key
```

| Key 类型 | 数量 | 用途 |
|---------|------|------|
| 免费账号 | 3 个（各 50 次/月） | 覆盖免费用户 3 次试用 |
| 付费账号 | 1 个（按需升级） | 免费额度耗尽后使用 |
