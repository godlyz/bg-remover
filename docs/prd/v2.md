# BGFree v2.0 产品需求文档

> 版本: v2.0  
> 日期: 2026-03-22  
> 状态: 待评审

---

## 一、项目背景

BGFree 是一个在线图片去背景工具，当前已完成 MVP 版本（v1.0），具备本地 AI 和云端 remove.bg 双引擎去背景能力。

v2.0 的目标是：**接入用户体系和支付系统，实现商业化运营。**

---

## 二、用户角色

| 角色 | 描述 |
|------|------|
| 未登录用户 | 未注册，只能使用本地去背景，云端 1 次试用 |
| 免费用户 | Google 登录后，本地无限 + 云端 3 次（终身） |
| 付费用户 | 订阅了入门/进阶/专业套餐，本地无限 + 云端按套餐额度 |

### 3.3 Google OAuth 登录

**已有，需调整：**

- 登录后首页显示用户头像和用户名
- 登录成功后跳回首页
- 未登录时点击云端处理，弹窗提示登录（而非直接跳 Google）
- 退出登录清除本地 session

### 3.4 用量计数

**云端处理时扣减逻辑：**

1. 未登录用户：云端 1 次（浏览器 localStorage 计数）
2. 已登录用户：
   - 免费额度：终身 3 次，扣完为止（D1 数据库记录）
   - 付费额度：按月重置（D1 数据库记录）
3. 用量不足时：弹窗提示升级套餐

**计数时机：** 云端处理请求发出时扣减（非处理成功后），防止超用。

### 3.5 个人中心

**入口：** Header 头像点击 → 下拉菜单 → 个人中心

**页面内容：**

| 模块 | 说明 |
|------|------|
| 用户信息 | 头像、用户名、Google 邮箱、注册时间 |
| 当前套餐 | 显示套餐名称 + 到期时间 |
| 用量统计 | 本月已用 / 总额度，进度条展示 |
| 升级/降级 | 套餐切换按钮（跳转定价页） |
| 付费历史 | 订阅记录列表（时间、套餐、金额、状态） |
| 取消订阅 | 退订按钮（当期有效，下月取消） |
| 账户管理 | 退出登录 |

### 3.6 定价页面

**入口：** Header "升级" 按钮 + 个人中心升级链接 + 用量不足弹窗

**页面结构：**

```
[定价页]

  标题: 选择适合你的方案
  副标题: 本地去背景永久免费，云端高质量处理按需订阅

  [免费]        [入门 $9.99]     [进阶 $29.99]    [专业 $79.99]
  3次(终身)      30次/月          100次/月          300次/月
  标清           高清             高清               超清
               WebP下载         WebP下载           WebP下载
                                历史20张           历史50张
                                                   历史100张
                [开始]          [开始]             [开始]

  FAQ 常见问题
```

**定价卡片交互：**
- 默认选中当前用户已有套餐（或进阶）
- 已有套餐的卡片显示 "当前方案" 标签
- 点击 [开始]：
  - 已登录 + 免费用户 → 跳转 PayPal 支付
  - 已登录 + 想升级 → PayPal 支付差额
  - 已登录 + 想降级 → 确认弹窗（下月生效）
  - 未登录 → 先跳转 Google 登录 → 再跳转支付

### 3.7 PayPal 支付

**支付流程：**

```
用户点击订阅 → 选择套餐 → PayPal 支付 → 支付成功回调 → 更新 D1 用户套餐和额度 → 跳转成功页
                                                          ↓ 失败
                                                      跳转失败页（重试按钮）
```

**技术方案：**

- PayPal Checkout（网页端标准支付）
- API Route `/api/payments/create-order` — 创建订单
- API Route `/api/payments/confirm` — 支付成功回调
- API Route `/api/payments/webhook` — PayPal webhook 通知
- 订阅模式：月付自动续费
- 用户可随时取消，取消后当期有效，下月降为免费

### 3.8 FAQ

**定价页底部展示，折叠/展开：**

| 问题 | 回答 |
|------|------|
| 本地去背景真的免费吗？ | 是的，永久免费，不限次数，无需注册 |
| 云端去背景和本地有什么区别？ | 本地在浏览器处理，速度快但质量一般；云端用专业 AI 处理，边缘更精细（头发丝、透明物体等） |
| 免费额度用完了怎么办？ | 注册后可获得 3 次云端免费体验，用完后可选择付费套餐 |
| 可以按年付吗？ | 目前仅支持月付，后续会开放年付优惠 |
| 如何取消订阅？ | 个人中心 → 取消订阅，取消后当期仍可使用，下月降为免费 |
| 支持哪些支付方式？ | PayPal（支持信用卡、借记卡、PayPal 余额） |
| 我的图片安全吗？ | 本地模式图片不会上传；云端模式通过加密传输，处理完即删除 |

### 3.9 未登录转化引导

**场景：** 未登录用户点击云端处理按钮时

**交互：**
1. 弹窗提示："注册后可获得 3 次云端免费体验"
2. 按钮：[Google 登录] [稍后再说]
3. 点 Google 登录 → 授权 → 跳回 → 自动开始云端处理

**场景：** 免费用户 3 次用完后

**交互：**
1. 弹窗提示："本月免费额度已用完，升级套餐继续使用云端高质量处理"
2. 按钮：[查看套餐] [继续用本地模式]
3. 点查看套餐 → 跳转定价页

## 四、非功能需求

### 4.1 数据存储

| 数据 | 存储方案 | 说明 |
|------|---------|------|
| 用户信息 | Cloudflare D1 | user_id, email, name, avatar, plan, created_at |
| 用量记录 | Cloudflare D1 | user_id, month, used_count, plan |
| 订阅记录 | Cloudflare D1 | user_id, plan, amount, status, start_date, end_date |
| 支付订单 | Cloudflare D1 | order_id, user_id, plan, amount, status, paypal_order_id |
| 未登录计数 | localStorage | browser_fingerprint → cloud_used_count |

### 4.2 remove.bg API Key 管理

**轮换策略：**

```
云端请求 → 检查所有 API Key 本月用量 → 选剩余最多的 Key → 转发请求
                                          ↓ 全部用完
                                    走付费订阅 Key
```

| Key 类型 | 数量 | 用途 |
|---------|------|------|
| 免费账号 | 3 个（各 50 次/月） | 优先消耗，覆盖免费用户体验 |
| 付费账号 | 1 个（按需升级） | 免费额度耗尽后使用 |

**存储：** Cloudflare KV 存储各 Key 的月用量计数，每月 1 日自动重置。

### 4.3 安全

- PayPal webhook 签名验证
- 支付回调幂等处理（防止重复扣费）
- 用户只能访问自己的数据
- API Key 不暴露给前端
